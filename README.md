# DeepSeek Chatbot

An AI chatbot built with FastAPI and DeepSeek model.

## Features
- Uses DeepSeek-R1-Distill-Qwen-1.5B
- WebSocket-based chat
- Stores conversations in PostgreSQL
- Docker support for easy deployment
